# TP0007/SEC-T-00001

## Scope

Der Test startet einen OWASP Scan gegen das installierte Panel.

## Beschreibung

Es wird ein Knoten angelegt, der ein Panel enthält. Die gesteuerte ZAP wird als OWASP Test verwendet.

## Skripte

- Start des OWASP Scanners: executer
